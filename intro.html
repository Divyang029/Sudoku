What is Sudoku?

Sudoku is a classic logic-based number puzzle. The goal is to fill a 9x9 grid with digits so that each column, each row, and each of the nine 3x3 sub-grids that compose the grid (also known as "regions") contains all of the digits from 1 to 9 without repetition.

How to Play Sudoku

  1. Each row must have all the numbers from 1 to 9 with no repetitions.
  2. Each column must have all the numbers from 1 to 9 with no repetitions.
  3. Each 3x3 sub-grid must have all the numbers from 1 to 9 with no repetitions.
  4. Start with a partially filled grid and use logic to fill in the missing numbers.
